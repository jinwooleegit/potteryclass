## 사용 방법
1. `create-icons.html` 파일을 브라우저에서 열어 로고와 파비콘 이미지 생성
2. 다운로드한 이미지를 다음과 같이 저장:
   - 루트 디렉토리에 `favicon.ico` 파일 저장
   - `images/icons` 폴더에 로고 이미지 저장
3. 생성한 후에는 `create-icons.html` 파일은 삭제해도 됨
