1. 사이트 개요

도자기공작소 숨 홈페이지는 체계적인 도예 교육 과정과 관련 정보를 제공하는 웹사이트입니다. 총 7개의 챕터로 구성되어 있으며, 도예의 기초부터 마케팅까지 전 과정을 다룹니다.

주요 구성

  • 챕터 1-7: 도예 교육과정의 핵심 내용을 단계별로 제공
  • 도예 용어집: 도예 관련 전문 용어 정리
  • 검색 기능: 원하는 정보를 빠르게 찾을 수 있는 검색 도구
  • 다크 모드: 사용자 환경에 맞는 화면 밝기 조절

2. 메뉴 및 탐색 방법

상단 메뉴를 통해 원하는 챕터나 페이지로 쉽게 이동할 수 있습니다.

상단 메뉴

모든 페이지 상단에 위치한 글로벌 메뉴를 통해 사이트의 주요 섹션으로 이동할 수 있습니다.

  • 홈: 메인 페이지로 돌아갑니다.
  • 챕터 1-7: 각 교육 단계별 내용으로 이동합니다.
  • 용어집: 도예 관련 용어를 찾아볼 수 있습니다.

모바일 메뉴

모바일 화면에서는 상단 우측의 햄버거 아이콘(☰)을 클릭하여 메뉴를 열 수 있습니다.

  1. 햄버거 아이콘을 탭합니다.
  2. 펼쳐진 메뉴에서 원하는 항목을 선택합니다.
  3. 다시 탭하면 메뉴가 닫힙니다.

탐색 팁

현재 보고 있는 페이지의 메뉴 항목은 강조 표시되어 있어 쉽게 구분할 수 있습니다.

3. 검색 기능 활용하기

원하는 정보를 빠르게 찾기 위해 상단 검색창을 활용하세요.

검색 방법

  1. 상단 메뉴의 검색 아이콘 또는 검색창을 클릭합니다.
  2. 찾고자 하는 키워드를 입력합니다 (예: "물레", "유약", "정형").
  3. 엔터키를 누르거나 검색 버튼을 클릭합니다.
  4. 검색 결과에서 원하는 항목을 선택하여 해당 페이지로 이동합니다.

효과적인 검색 팁

  • 정확한 키워드 사용: 가능한 구체적인 용어를 사용하면 더 정확한 결과를 얻을 수 있습니다.
  • 여러 키워드 시도: 원하는 결과가 나오지 않으면 유사한 다른 키워드로 검색해보세요.
  • 앵커 링크 활용: 검색 결과에서 특정 섹션으로 바로 이동할 수 있는 링크를 제공합니다.

4. 챕터 내 이동하기

각 챕터 페이지는 여러 섹션으로 구성되어 있으며, 효율적으로 탐색할 수 있는 방법이 있습니다.

헤더 링크

큰 제목(h2)을 클릭하면 해당 섹션의 고유 링크가 복사되어, 특정 섹션을 공유하는 데 유용합니다.

브레드크럼 네비게이션

페이지 상단의 경로 표시기(홈 > 현재 페이지)를 통해 현재 위치를 확인하고 이전 페이지로 쉽게 돌아갈 수 있습니다.

챕터 내 구성

각 챕터는 일반적으로 다음과 같은 구조로 되어 있습니다:

  • 개요: 챕터에서 다루는 주제에 대한 소개
  • 주요 섹션: 세부 주제별로 나뉜 학습 내용
  • 실습 과제: 배운 내용을 적용해볼 수 있는 활동
  • 관련 자료: 추가 학습을 위한 링크와 자료

5. 다크 모드 사용하기

눈의 피로를 줄이고 배터리를 절약하기 위해 다크 모드를 활용할 수 있습니다.

다크 모드 전환 방법

  1. 페이지 상단 메뉴의 오른쪽에 있는 다크 모드 아이콘(🌙/☀️)을 클릭합니다.
  2. 클릭할 때마다 라이트 모드와 다크 모드가 전환됩니다.
  3. 사용자의 선호도는 브라우저에 저장되어 다음 방문 시에도 유지됩니다.

자동 모드 전환

기본적으로 사이트는 사용자의 시스템 설정(운영 체제의 다크 모드 설정)을 따릅니다. 수동으로 전환하면 이 설정이 우선합니다.

6. 접근성 기능

도자기공작소 숨 홈페이지는 다양한 사용자가 편리하게 이용할 수 있도록 접근성 기능을 제공합니다.

키보드 탐색

키보드만으로도 모든 메뉴와 링크를 탐색할 수 있습니다. Tab 키를 사용하여 요소 간 이동이 가능합니다.

텍스트 크기 조정

브라우저의 텍스트 크기 조절 기능(보통 Ctrl + 또는 Ctrl -)을 사용하여 콘텐츠의 크기를 조절할 수 있습니다.

스크린 리더 호환성

모든 이미지에는 대체 텍스트가 제공되며, 의미 있는 마크업 구조를 통해 스크린 리더 사용자도 콘텐츠를 쉽게 이해할 수 있습니다.

7. 모바일 사용자를 위한 팁

도자기공작소 숨 홈페이지는 모바일 환경에 최적화되어 있습니다. 다음은 모바일에서 사이트를 효과적으로 사용하는 팁입니다.

모바일 최적화 기능

  • 반응형 레이아웃: 화면 크기에 맞게 콘텐츠가 자동으로 조정됩니다.
  • 가로 스크롤 최소화: 모든 콘텐츠는 모바일 화면에 맞게 최적화되어 가로 스크롤이 거의 필요 없습니다.
  • 터치 친화적 요소: 버튼과 링크는 터치하기 쉬운 크기로 제공됩니다.
  • 빠른 로딩: 이미지 최적화를 통해 모바일 환경에서도 빠른 페이지 로딩이 가능합니다.

오프라인 읽기 팁

중요한 페이지는 모바일 브라우저의 '페이지 저장' 기능을 사용하여 오프라인에서도 볼 수 있도록 저장할 수 있습니다.

8. 문제 해결

사이트 사용 중 문제가 발생할 경우 다음 해결 방법을 시도해 보세요.

일반적인 문제

  • 페이지가 로드되지 않음: 인터넷 연결을 확인하고 브라우저를 새로고침하세요.
  • 이미지가 표시되지 않음: 브라우저 캐시를 지우고 다시 시도하세요.
  • 메뉴가 작동하지 않음: 최신 브라우저를 사용 중인지 확인하세요.

해결 단계

  1. 브라우저 새로고침 (F5 또는 Ctrl+R)
  2. 다른 브라우저로 시도
  3. 기기 재시작
  4. 브라우저 캐시 및 쿠키 삭제

지원 문의

문제가 지속될 경우 milgae@naver.com으로 문의해 주세요. 이슈를 상세히 설명하고 가능하다면 스크린샷을 함께 첨부해 주시면 도움이 됩니다.
